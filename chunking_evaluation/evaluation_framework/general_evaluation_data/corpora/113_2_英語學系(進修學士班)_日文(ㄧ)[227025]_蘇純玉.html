課程目標   Learning Objectives
1.本課程著重於正確地以東京標準音發音，讓學生確實地對話，藉由小組對話與討論，克服說日文的恐懼。能夠充分理解與運用每單元之句型，進而提高學習日語之興趣與培養自然表達基礎生活日語之能力。
2.此課程不僅啟發日常會話的溝通技巧，還學習簡單自我介紹、日常寒暄、指示詞的運用(人/物的存在位置)、簡易購物與金額的表達等基礎日語表達能力，期待將所學運用於實際日常生活中。
3.同時適時地加入相關文法內容的影片觀賞為輔助教材，引發學習興趣；藉由輔助教材認識日本風俗民情同時更加強課程學習內容文型運用表達之能力。 

先修 ( 前置 ) 課程   Prerequisite
1.對『日文』與『日本文化』抱持「積極主動學習」之態度者。
2.日語程度已經超出此課程預計之學習範圍者，亦不適合選修此課程，請將此選修機會讓與其他尚未學過此課程內容者。
3.「一分耕耘、一分收穫」，只欲獲得學分而不願付出相對努力者，請勿選修此課程。
4.眾所皆知「日文」是極受喜愛選修之第二外語，欲選修人數眾多，若是抱持「先選再說，不喜歡屆時再辦理棄選」之態度者，請勿選修此課程。
5.此課程著重「對話練習」，因此拒絕句型會話演練者，亦不適合選修此課程。
6.各項大、小考試應視為”如指考、學測般重要”，請於規定時間應考，如有缺考以零分計算，不得要求補考或更改考試時間。
7.課堂中"嚴格禁止睡覺，滑手機"。一經發現違規者，"出・缺席狀況及課堂表現20%" 零分計算，不得有所異議。
