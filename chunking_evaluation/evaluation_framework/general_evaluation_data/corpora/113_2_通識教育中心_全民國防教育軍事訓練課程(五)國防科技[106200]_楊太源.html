課程目標   Learning Objectives
1.激勵學生愛國情操，認識當前國家處境及長治久安之道。
2.能說明國防科技對國家安全的影響，以及對我們個人所產生的影響。
2.使學生能了解國防科技，充實學生軍事知能。
3.使學生了解實際我國執行國防科技政策的組織及單位。
4.闡述現代戰爭中聯合作戰的重要性及人才培育的必要性，進而深切體認國防科技。
 

先修 ( 前置 ) 課程   Prerequisite
（一）利用課餘時間多加蒐集各課程單元相關資料，並隨時與其他授課教官相互討論研析，以充實教學資源，達到交流共享。
（二）閱讀有關國防科技、軍事武器、時事報導等相關新聞雜誌、網站，以隨時補充新知。
