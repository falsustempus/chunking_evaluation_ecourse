課程目標   Learning Objectives
1.了解新課綱國中綜合活動學習領域的理念、目標與、學習內容和學習表現。
2.熟練綜合活動學習領域的課程發展、設計、教案撰寫要領與多元評量。
3.熟練實施綜合活動單元活動設計之教學教法。
4.能遵守教學倫理。 

先修 ( 前置 ) 課程   Prerequisite
1.此課程為實務性課程，需先修習綜合活動領域概論。
2.此課程採實務能力實作導向教學，從他人的教案評述開始、至自已的教案設計與試教到期末的模擬教師徵選的教學演練，重視學生教學能力的循序漸進培養。
3.本課程邀請國中、高中輔導活動課教師進行座談會分享實務現場。
4.正式之教學內容在開學第一次上課與同學討論後，再行修訂。

成績評量方式補充說明   
1.分組教案評述說明:1~2人一組，找到一個你覺得很棒的教案，進行此教案的介紹，並加以評論其優缺點，以及如果是你們設計，你可能會做什麼修改。時間約30分鐘。可參考教科書第175頁表9-1和9-2。
2.分組主題試教說明:1~2人一組，依指定主題設計一個教案並進行試教其中一節（45分鐘），其它同學扮演現場學生。可參考教科書第165頁教案範例。
3.期末主題試教說明:設計一個教案並進行試教（20分鐘）模擬教甄。
4.國中與高中教學現場教師座談，需要請同學事前彙整想要討論的問題。
