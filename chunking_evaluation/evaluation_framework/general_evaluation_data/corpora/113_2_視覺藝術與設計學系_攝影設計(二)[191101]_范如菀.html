課程目標   Learning Objectives
以攝影為基礎出發點,在視覺藝術中攝影所佔用的視覺表達方式呈現之分析，課堂中會介紹攝影作品於設計當中佔有的比例與影響。
瞭解攝影對於設計作品中所佔有的成份比例及其重要性。
並重視實際操作相機製作影像，最終目標是以攝影作品輔助設計完成最終視覺表現作品為主。
(實際操作有基本器材需求,需自行準備專業單眼相機及筆電（需自行下載安裝軟體photoshop）。
本課程以實際拍攝為主，實拍課程結束後會安排檢討拍攝內容及給予拍攝修正建議。
本學期於第十五週需舉辦成果發表，選修同學需配合籌劃與布展等工作項目，並製作作品協助展出。
期末需繳交作品，亦需自行準備作品所需輸出與製作費用，無法接受者請勿選修本門課程。 

先修 ( 前置 ) 課程   Prerequisite
攝影設計(一)、色彩學。

必須選修以上三門課程共四學分，方可選修本課程。
未選修攝影設計（一）的同學，建議先先選修（一），再選修（二）。
本課程不重複教學基礎相機操作。

外系同學亦請先選修以上課程,方可選修本課程。
