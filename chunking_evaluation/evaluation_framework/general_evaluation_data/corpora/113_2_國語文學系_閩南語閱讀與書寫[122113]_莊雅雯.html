課程目標   Learning Objectives
本課程以台語散文為主，藉由文章培養學生閱讀台語文的能力、加強對台語漢字的認識，並強化台語拼寫能力，以期提升閩南語閱讀書寫能力。以此基礎進一步瞭解台灣文化，使學生從中對台灣在地文化有深入的瞭解、尊重與關懷。希望藉由本課程的開設達到以下目標：
1.培養學生閱讀台語文的能力。
2.訓練學生台語文的書寫能力。
3.教育部台灣羅馬字拼音練習。
4.深化學生在地文化素養。
5.多面向的台語文教學與學習。

 

先修 ( 前置 ) 課程   Prerequisite
修過閩南語概論課程，或有台灣羅馬字拼音基本概念。
