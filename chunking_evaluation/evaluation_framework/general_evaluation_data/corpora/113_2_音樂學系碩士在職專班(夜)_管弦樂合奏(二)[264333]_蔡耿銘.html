課程目標   Learning Objectives
本課程目標為讓碩士生能與校外樂團有合作實習演出之經驗，以開拓其視野與增進專業演出能力。
 

先修 ( 前置 ) 課程   Prerequisite
本課程得符合當期樂團演出編制與通過指導老師審核方可修習。
