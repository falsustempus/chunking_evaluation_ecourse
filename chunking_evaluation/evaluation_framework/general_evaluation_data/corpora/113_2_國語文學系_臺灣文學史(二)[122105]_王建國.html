課程綱要
一、教學目標：
0. 不論你是哪裡人，也不論你的認同，既然生在/身在這片土地上，不妨來瞭解土地上的那些年那些事。
1. 臺灣文學 / 臺南文學重要篇章選讀。
2. 認識及瞭解各時期重要臺灣文學 / 臺南文學 作家之作品風格與特色。
3. 認識臺南及建構各自故鄉（依修課同學出生、成長的故鄉而有所不同）文學史的發展脈絡，並產生積極性的交流與對話，進而思辨其與臺灣文學史的脈絡關係。
4. 瞭解臺灣文學史的發展脈絡，進而比較各家(國)臺灣文學史的異同優劣。

二、教材內容:
以臺灣 / 臺南文學史為主軸，並選讀重要文學作品、參酌相關文獻史料、專著、影片及論文篇章。

三、實施方法：
講授、課堂討論、影片暨紀錄片欣賞、個別或分組書面報告、走讀及漫遊臺灣 / 臺南文學地景、快閃演出……等。

「如何文學，怎樣臺南」：「一府、二鹿、三艋舺」，正標誌著臺灣歷史文化的發展軌跡，臺南府城的重要性，不言可喻。本課程主要以漢文學（書面文學）發展為主，內容涵蓋古典文學與現代文學的範疇，並涉及重要歷史人物、事件、文學（思潮）主題、近現代文學作家與團體……等。尤其，臺南文學與臺灣文學的發展關係密切，具有（臺南文學 / 臺灣文學）雙軌並行與相互辯證的意義，故本課程將以臺南文學史 / 臺灣文學史為脈絡，介紹臺南文學 / 臺灣文學的發展。

期中作業與期末分組成果發表（以下三擇一）：一、選註並賞析《全臺詩》具有「臺灣國風」意涵的「臺灣竹枝詞」詩作。二、「君自故鄉來，應知故鄉事」－－修課同學依清領時期臺灣行政區分組，期中前先行獨立完成介紹自己故鄉的古典文學家及其作品（2000~3000字），期末前再嘗試與其他同縣市同學互相討論、協力整合（分組），進一步完成故鄉文學地景踏查之統整性規劃（12000字）。三、街頭快閃：「大天后宮」前（原寧靖王府）演出寧靖王與五位妾媵故事（10~15分鐘。

＊額外加分(以下三選一)：
1.選擇明鄭(南明)時期詩人詩文(以一首 / 篇為主)，或擇定一位來台文人之渡海詩文(以一首 / 篇為主)，對其人其作加以介紹、註釋(包含生字(詞)、典故、考證)、賞析，繳交完整書面報告(字數：1200~1500字)；作品來源可參考：施懿琳等編撰，《全臺詩》(第一~十二冊)，台南：國家臺灣文學館，2004。(*請事先與老師商定或討論其人其作，謝謝)
2.閱讀齊邦媛，《巨流河》(臺北市:天下遠見，2009年)；並撰寫讀書心得報告(字數：1200~1500字)。
