課程目標   Learning Objectives
1. 以藝術家精神為本質, 不僅著重個人彈奏技巧, 進而強調與他人的合作, 將音樂融為一體
2. 廣泛的曲目從古典到現代戲劇舞蹈, 建立學生對其他樂器的曲目認識
3. 加強讀譜能力
4. 強化學生伴奏專業能力，與音樂創作能力
 

先修 ( 前置 ) 課程   Prerequisite
伴奏法 (一) 

彈性教學規劃   Flexible Teaching/Planning Schedules
＊本課程實施16+2週彈性教學方案，其中第17、18週之彈性規劃如下：
展演實作
自主學習
線上討論與分享交流
