課程目標   Learning Objectives
本課程介紹攝影藝術的表達方式與拍攝實務。由光圈、快門、感光度的原理與運用起始，加上構圖與採光。內容包含生活中各式題材的拍攝方式及作品賞析。期使同學能夠藉由本課程之學習，享受數位攝影在生活記錄上的便利性，培養敏銳觀察力、提升美學涵養，甚至帶來藝術創作上的啟發。 

先修 ( 前置 ) 課程   Prerequisite
無先修課程，老師會從基礎講起；但同學需對攝影技術或影像欣賞具有興趣，方才建議選修此課程。老師會於必要時提供數位單眼相機讓同學操作練習。
