課程目標   Learning Objectives
了解現今桌球技術，平時培養獨立判斷與應變的能力，進而學習運用在比賽之中。養成團隊當中的互助合作、服從、守法、公平、公正的精神。學會將桌球作為生涯規劃之一。 

先修 ( 前置 ) 課程   Prerequisite
此課程是給予以桌球獨招進來的學生(桌球校隊)修習，必須修滿4年，一般生無法選修。
