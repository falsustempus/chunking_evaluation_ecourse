課程目標   Learning Objectives
知識:了解基本犯罪與法律知識，藉案例分析強化學生的法學素養。

技能:學習日常生活中遇到犯罪與法律問題的解決問題的能力。

情意:透過參訪或分組討論以增加表達能力，對犯罪與法律問題之解決有臨場的表現與經驗。

人際關係:了解複雜的犯罪及法律關係，使免於因不知法律而犯罪，以降低衝突，增進人際關係

教導學生了解我國當前的犯罪現象及各項刑事政策與法律規定，進而比較、分析，使在生活當中碰到問題能了解相關規定，而避免觸法或得到充分的了解與解決法律與社會問題。 

先修 ( 前置 ) 課程   Prerequisite
學生必須先思考與了解台灣目前的犯罪狀況,並進而思考如何透過刑事政策的制定與策略來達成犯罪的防治與預防,以使整個國家更安定與繁榮.
教導學生了解我國當前的犯罪現象及各項刑事政策(尤其有關死刑存廢政策)與法律規定，進而比較、分析，使在生活當中碰到問題能了解相關規定，而避免觸法或得到充分的了解與解決法律與社會問題。
