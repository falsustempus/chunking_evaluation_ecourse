課程目標   Learning Objectives
1. 了解工程問題與所對應的數學模型(B, D, E)
2. 熟悉向量微分與積分計算(B, E)
3. 熟悉傅立葉分析(B, D, E)
4. 熟悉偏微分方程式解法、解的特性與其在工程上之應用(B, E)
5. 培養同學具有數學分析能力。(B, C, D, E)
