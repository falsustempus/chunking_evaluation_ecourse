課程目標   Learning Objectives
1. 培養學生對鋼琴音樂的興趣與廣泛認識
2. 除了認知上的學習,更重於對音樂的欣賞能力與正確的演奏詮釋觀念
3. 對鍵盤樂器的廣泛認識, 有助於日後的音樂學習與教學
 

先修 ( 前置 ) 課程   Prerequisite
鋼琴作品研究(上) 

彈性教學規劃   Flexible Teaching/Planning Schedules
＊本課程實施16+2週彈性教學方案，其中第17、18週之彈性規劃如下：
線上教學/討論
自主學習
學術研究與寫作
