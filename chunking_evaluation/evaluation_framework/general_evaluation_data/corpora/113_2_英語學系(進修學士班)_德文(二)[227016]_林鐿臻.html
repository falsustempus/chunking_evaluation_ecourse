課程目標   Learning Objectives
繼續強化學生德語字母、字及句子的正確發音基礎；以日常生活常出現之狀況做為會話主題, 分小組對談，訓練學生勇於開口，除了基本的自我介紹的能力，進一步藉由不同的情境主題擴大詞彙量。並讓學生對德國人生活的點滴有初步認識。以學生已學之德文為基礎, 經由日常生活對話, 相關練習和簡單的實用文章閱讀, 培養學生日常基礎的德語聽說讀寫能力, 學生除學習德語之外, 也將於課堂中認識德語系國家之城市、風土民情與文化, 導引正確使用德語字典, 培養基本的德文造句寫作能力, 習得近1000至2000個德語字彙, 了解德國, 奧地利瑞士三國的文化, 教學內容涵蓋日常生活主題, 自我與他人介紹, 居家生活, 飲食. 價值, 評斷, 休閒生活, 旅遊。
 

先修 ( 前置 ) 課程   Prerequisite
1.本課程為初階課程。
2.因開口練習德文的機會相對少，因此本課程重視「對話練習」，因此課堂上若無法做對話練習者，
則不適合本課程。
3.請假務必事先告知,並按照規定請假。
