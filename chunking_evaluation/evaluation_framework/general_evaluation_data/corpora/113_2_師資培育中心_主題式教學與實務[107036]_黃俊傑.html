課程目標   Learning Objectives
1.學生統整所學過的課程與教學理論，及課程設計的知識與理念。
2.聚焦於主題式教學的設計與實作，讓學生能依據十二年國教課綱的精神發展主題式課程，設計合適的主題課程。 

先修 ( 前置 ) 課程   Prerequisite
1.已修習並通過「教學原理」、「課程原理」等課程，並精熟該課程的專業知能。
2.建議已修習並通過「課程發展與設計」、「教學設計與發展」、「跨領域課程理論與實務」及教材教法等相關課程，並精熟該課程的專業知能。
