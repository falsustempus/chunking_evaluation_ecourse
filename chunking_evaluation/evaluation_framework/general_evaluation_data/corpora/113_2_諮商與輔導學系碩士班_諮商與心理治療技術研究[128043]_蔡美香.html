課程目標   Learning Objectives
1.藉由諮商與心理治療技術之學習及演練，了解助人技巧及助人歷程。
2. 透過助人技巧之探究與問題研討，精進諮商技術。
3. 探討相關實徵研究之議題，增進該議題之諮商專業知能及可能之限制。
4. 藉由演練及實際諮商個案之過程，增進諮商技巧。
5. 藉由課程訓練培養學生成為專業諮商心理師。
