課程目標   Learning Objectives
本課程將先介紹運算思維，這是一種用電腦的邏輯來解決問題的思維。主要有四種核心能力:
Decomposition, Pattern recognition, abstraction, algorithm design.

進而簡單介紹程式設計的三種基本控制流程：循序、選擇與迴圈。

最後以常見的演算法為例來統整的介紹運算思維與程式設計。
