課程目標   Learning Objectives
1.研析教學原理的意義及內容。
2.探討教學原理的理論基礎。
3.深究教學原理的主要內涵。
4.探究教學原理的應用。
 

先修 ( 前置 ) 課程   Prerequisite
1.請在上課前依據進度作預習工作。
2.如不克前來上課請事先請假。
3.請在上課前代為準備各種教學媒體。
4.指定作業報告請用單槍投影機及手提電腦。
5.請如期完成指定作業以方便課堂討論。
