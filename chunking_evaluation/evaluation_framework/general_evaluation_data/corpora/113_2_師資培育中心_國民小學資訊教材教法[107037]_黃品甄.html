課程目標   Learning Objectives
1.引導師資生探究國內資訊教育的重要發展方向與實施
2.引導學生探究小學生的資訊素養
3.引導師資生探究小學資訊教師應具備的資訊素養
4.引導師資生設計資訊教材與教案
5.引導師資生從試教中反思與精進教學能力 

先修 ( 前置 ) 課程   Prerequisite
