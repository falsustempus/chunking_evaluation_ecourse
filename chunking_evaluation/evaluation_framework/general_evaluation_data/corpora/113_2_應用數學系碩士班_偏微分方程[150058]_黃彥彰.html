課程目標   Learning Objectives
本課程為研究所偏微分方程基礎課程，亦可視為大學部分析學(Analysis)和微分方程課程的延伸, 主要使用partial differential equations / Lawrence C. Evans, 1998 這本書，外加部分的補充資料(幾何分析教材).  

先修 ( 前置 ) 課程   Prerequisite
線性代數、微積分、高等微積分、微分方程
