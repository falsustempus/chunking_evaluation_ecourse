課程目標   Learning Objectives
了解手機平台發展趨勢日趨重要，行動網路應用服務日新月異，帶領學生學習手機程式的應用與限制，快速創作手機 APP，不需太深入的程式語言基礎。類似圖框排列的方式，一步步引領且透過拖曳方式寫出APP程式，引發學生對程式設計的興趣，訓練學生的邏輯思考與程式設計基礎概念，進行智慧型手持裝置上的應用程式開發與驗證，解決生活上的問題，讓學生藉由學習行動裝置程式設計的入門課程，學習具備基本邏輯思考能力。 

先修 ( 前置 ) 課程   Prerequisite
學生須具備基本邏輯思考與資訊科技能力，課程正式上課後，請同學隨時注意課程公告訊息。
