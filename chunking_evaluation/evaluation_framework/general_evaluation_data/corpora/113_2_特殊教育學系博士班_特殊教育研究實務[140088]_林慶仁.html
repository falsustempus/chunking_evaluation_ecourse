01 系所 能具備專業理論知識，深入瞭解各類特殊需求者特質及需求，並積極參與特殊教育及輔助科技相關活動，以培養適性揚才的理念與態度 0
02 系所 能有效利用適當的評量工具與方法，了解特殊需求者的現況能力與需求，訂定個別化教育、輔導、服務等相關計畫 0
03 系所 能因應特殊需求者個別需求，設計課程、教材（輔具）、教學活動及服務內容 0
04 系所 能有效統整教學理論、方法、輔助科技與輔導策略，並與其他相關專業人員合作，營造正像環境，以提升特殊需求者學習潛能與生活品質 0
05 系所 能具有溝通協調能力，與家長、教師、行政人員及其他相關人員建立良好互動與合作關係 0
06 系所 能掌握特殊教育（輔助科技）的脈動，瞭解相關的議題，提供適當的資訊，以促進特殊需求者之權益 0
07 系所 能具備特殊教育（輔助科技）研究的知能與興趣，探索相關問題，創新教學與應用方法，以解決教學及實務問題 0
08 系所 能具備外文閱讀能力，積極參與國內外相關學術活動及學術團體 0
