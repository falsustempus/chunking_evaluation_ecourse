課程目標   Learning Objectives
本課程 徵求以下樂器編制：
- 薩克斯風 Saxophone 5名 (長笛可)
- 小號 Trumpet 5名
- 長號 Trombone 4名
- 鋼琴 Piano 2名
- 電貝斯（E.Bass） 或 低音提琴（Double bass）
- 爵士鼓 Drums 2名
- 打擊（鐵琴）Percussion 1名
- 小提琴 Violin 4-5名

每學期有期末音樂會


【課程目標】
一、了解並活用爵士樂語法（articulation）與演奏法的不同，進而啟發更多元的演奏能力，以期能夠更加表現音樂的內涵。
二、主要學習節奏風格:搖擺 Swing、巴莎諾瓦 Bossanova、拉丁Latin、放克Funk。 

先修 ( 前置 ) 課程   Prerequisite
- 無先修前置課程需要 。
- 需具備3年以上樂器能力與五線譜基本視譜能力。
- 本課程非常要求【出席率】與【課堂參與度】，常態性缺曠的同學請勿選修此課程。
