課程目標   Learning Objectives
金屬箔常為東方藝術表現的材料之ㄧ，除一般平貼技法外，更有盛上堆金、灑金等各式技法。經由紙本線描及礦物彩繪之技法的運用與練習，能夠讓學生深入了解製作過程，進而思考幫助學生在創作上呈現東方媒材的多樣性。 

先修 ( 前置 ) 課程   Prerequisite
讓學生能了解膠彩繪畫材料及技法之運用，經由實際講解及操作示範，經由膠礬水刷染到紙本線描及礦物彩繪之技法應用與練習，能夠讓學生深入了解製作過程，進而思考幫助學生在創作上呈現東方媒材的多樣性
