課程目標   Learning Objectives
This course is for students with little or no programming experience. It aims to provide students with an understanding of the role computation can play in solving problems. It also aims to help students to feel justifiably confident of their ability to write small programs that allow them to accomplish useful goals. The class will use the C programming language. Moreover, this course will introduce basic concepts of data structures and object oriented programming.
