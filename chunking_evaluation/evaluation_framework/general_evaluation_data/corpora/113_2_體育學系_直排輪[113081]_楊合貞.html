課程目標   Learning Objectives
1.體驗直排輪運動的樂趣
2.透過直排輪運動，促進身體協調性與靈活性
3.了解與熱愛直排輪運動
4.熟悉直輪輪運動的各項技巧 

先修 ( 前置 ) 課程   Prerequisite
1.適合運動的服裝
2.自行準備直排輪裝備
