課程目標   Learning Objectives
1. 認識公文與文書處理在機構內部的程序與簽辦實務。
2. 具備公文閱讀與理解能力。
3. 具備撰擬公文（函文、簽稿、開會通知單）等公務處理能力，提高就業力。
4. 透過案例教學法提升行政實務素養，強化學用連結、問題解決與批判思考能力，培養優質公務與教育行政人力。
5.促進SDGs4（優質教育：確保有教無類、公平以及高品質的教育，及提倡終身學習）目標的達成。

教科書或參考用書   Textbooks or Reference Books
館藏書名   Library Books
備註   Remarks
1. 以自編教材為主：：政府組織與新聞素養、文書處理與檔案管理手冊重點摘要（行政院秘書處）、政府文書格式參考規範（國家發展委員會）、公文試題解析、外部來文/收發文作業/簽辦公文、內部創文等簽辦作業、案例教學（我是「承辦人」公文實作）等。
2.其他參考資料：
（1）高全（2016）。絕對制霸 公文－國考速成秘笈。考用出版社。
（2）應用中文編著委員會編（2008）。應用中文。麗文。
（3）邱忠民（2008）。公文製作原理與實作。麗文。
