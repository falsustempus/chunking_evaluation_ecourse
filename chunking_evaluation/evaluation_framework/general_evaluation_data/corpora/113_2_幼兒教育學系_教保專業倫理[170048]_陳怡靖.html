課程目標   Learning Objectives

1.了解幼兒教保專業倫理之意義及內涵。
2. 熟悉教保服務人員之權責及倫理規範。
3. 了解與幼兒教保工作相關之重要政策及法規。
4. 探討幼兒園相關倫理議題，並培養專業判斷及實踐能力。
5. 學習教保專業人員情緒管理及人際溝通。
