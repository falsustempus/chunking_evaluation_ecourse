課程目標   Learning Objectives
本課程採實務導向，介紹如何使用 Python 程式語言進行資料分析。

1. 熟悉 Python 程式語言以及資料分析相關套件的使用方式。
2. 瞭解機器學習的基本概念以及不同的機器學習方法的使用。
3. 應用本課程所學於分析實務資料。 

先修 ( 前置 ) 課程   Prerequisite
無，唯修過程式語言（Python基礎語法）者佳。
