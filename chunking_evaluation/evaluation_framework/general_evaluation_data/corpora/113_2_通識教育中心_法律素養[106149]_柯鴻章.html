課程目標   Learning Objectives
知識:了解基本法律知識，藉案例分析強化學生的法學素養。

技能:學習日常生活中遇到法律問題解決問題的能力。

情意:透過參訪或分組討論以增加表達能力，對法律問題之解決有臨場的表現與經驗。

人際關係:了解複雜的法規及法律關係，使免於因不知法律而引起糾紛，以降低衝突，增進人際關係
 

先修 ( 前置 ) 課程   Prerequisite
課程介紹及上課要求事項、各項評量方式及標準，我國司法制度簡介及與司法體系發展史等概括性的介紹，使同學能對我國司法各種體系及主要的民刑訴訟有一概念與全面性的了解，以便進入具體的法律事件或案例解析時迅速的知解其內容與意義，提升法律的素養。
