課程目標   Learning Objectives
透過投影片與解說了解桌球的起源與基本知識。經由示範與實際操作熟習桌球初中級基本動作，以及學會單雙打比賽規則。最終培養對桌球運動的興趣，以及在未來教學上可用的教學方法。 

先修 ( 前置 ) 課程   Prerequisite
請自備好一點的球拍，穿著運動服與運動鞋。
